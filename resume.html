SVM, RFC, PCA

The project implements Support Vector Machines (SVM) and Random Forest Classifier (RFC) on the Obesity Level dataset from Kaggle to analyze classification performance. Initial preprocessing prepares the data for modeling, and results from SVM and RFC are compared using accuracy, precision, and recall metrics. Random Forest's Out-of-Bag (OOB) score assesses model robustness. Principal Component Analysis (PCA) is then applied for dimensionality reduction, simplifying the feature space while preserving variance. Post-PCA performance of both classifiers is evaluated, highlighting trade-offs between computational efficiency and classification accuracy. The analysis emphasizes the significance of dimensionality reduction in handling complex datasets..
